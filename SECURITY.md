# Security Policy

## Reporting a Vulnerability

trickle ships pure-CSS-and-React components with no runtime dependencies that could carry vulnerabilities of consequence — but if you find a security issue (e.g. an XSS surface in how text props are rendered), please report it privately rather than filing a public issue.

Use **GitHub Security Advisories** on this repo: https://github.com/trickle-ui/trickle/security/advisories/new

Or email: `security@tricklekit.dev`

Please do not disclose publicly until a fix is available.

## Supported Versions

The latest tagged release is supported. Older versions receive no patches — you're encouraged to re-run `npx shadcn add` to pull the latest source for any component.
